# ميزة التوليد الذكي للمحتوى

تم إضافة ميزة توليد محتوى الطلب تلقائياً باستخدام الذكاء الاصطناعي.

## التثبيت

1. احصل على مفتاح API من Groq (مجاني):
   - اذهب إلى https://console.groq.com
   - قم بالتسجيل أو تسجيل الدخول
   - انسخ API key

2. أنشئ ملف `.env.local` في المجلد الجذر:
   ```bash
   GROQ_API_KEY=your_api_key_here
   ```

3. أعد تشغيل التطبيق:
   ```bash
   npm run dev
   ```

## الاستخدام

1. أدخل الموضوع (مثال: "طلب المشاركة في مسابقة توظيف عون إدارة رئيسي")
2. اضغط على زر "✨ توليد ذكي"
3. سيتم توليد محتوى احترافي تلقائياً بناءً على الموضوع
4. يمكنك تعديل المحتوى يدوياً إذا أردت

## الخصائص

- توليد محتوى احترافي بالعربية الفصحى
- يتناسب مع الموضوع المدخل
- يتضمن الصيغ الرسمية والموقرة
- سريع ومجاني

## الخوادم المدعومة

- **Groq** (مجاني) - السرعة 100+ tokens/sec
- **OpenAI** (مدفوع) - يمكن إضافته لاحقاً

## استكشاف الأخطاء

إذا لم يعمل التوليد:
1. تأكد من تعيين `GROQ_API_KEY` في `.env.local`
2. تأكد من أن API key صحيح من Groq console
3. تحقق من اتصال الإنترنت
4. اطلع على أخطاء المتصفح في Console (F12)
